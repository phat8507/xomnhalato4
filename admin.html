Memory Lane Xem website
Trang hướng dẫn cho người mới

Chỉ cần sửa data.js và upload file vào đúng thư mục.

Website này không còn dùng localStorage. Từ bây giờ mọi người mở link đều nhìn thấy cùng một nội dung, vì ảnh, video và dữ liệu đều nằm trực tiếp trong repository GitHub.

1. Upload file Ảnh vào photos/, video vào videos/
2. Mở data.js Sửa tiêu đề, mô tả, caption và tên file
3. Commit changes GitHub Pages sẽ tự cập nhật sau ít phút

File bạn sẽ sửa thường xuyên

  • data.js: sửa toàn bộ nội dung
  • photos/: thêm ảnh mới
  • videos/: thêm video mới

Quy tắc nhớ nhanh

Tên file trong data.js phải giống chính xác tên file bạn upload lên repository.

01

Cách upload ảnh vào photos/

  1. Mở repository trên GitHub.
  2. Vào thư mục photos.
  3. Bấm Add file rồi chọn Upload files.
  4. Kéo ảnh vào hoặc chọn từ máy tính.
  5. Bấm Commit changes.
02

Cách upload video vào videos/

  1. Vào thư mục videos trong repository.
  2. Bấm Add file rồi chọn Upload files.
  3. Chọn file .mp4 từ máy tính.
  4. Bấm Commit changes để lưu.
03

Cách mở và sửa data.js

Trong repository, mở file data.js rồi bấm biểu tượng cây bút để chỉnh sửa trực tiếp trên GitHub.

const siteData = {
  siteTitle: "Những Năm Tháng Cấp 3",
  schoolName: "THPT ...",
  years: "2021 - 2024",
  memories: [
    {
      title: "Ngày khai giảng",
      description: "Một vài dòng mô tả",
      cards: [
        {
          type: "photo",
          src: "photos/khai-giang.jpg",
          caption: "Chúng mình ngày ấy"
        }
      ]
    }
  ]
};
04

Đổi tiêu đề website, tên trường, năm học

Bạn chỉ cần sửa 3 dòng đầu trong data.js:

siteTitle: "Những Năm Tháng Cấp 3",
schoolName: "THPT Nguyễn Du",
years: "2021 - 2024",
05

Thêm một kỷ niệm mới

Trong mảng memories, thêm một khối mới theo mẫu sau:

{
  title: "Ngày bế giảng",
  description: "Một vài dòng mô tả cho kỷ niệm này",
  cards: [
    {
      type: "photo",
      src: "photos/be-giang-1.jpg",
      caption: "Tấm ảnh mình thích nhất"
    },
    {
      type: "video",
      src: "videos/be-giang.mp4",
      caption: "Khoảnh khắc đáng nhớ"
    }
  ]
}
06

Thêm một ảnh mới vào một kỷ niệm đang có

  1. Upload ảnh vào thư mục photos/.
  2. Mở data.js.
  3. Tìm đúng kỷ niệm bạn muốn sửa.
  4. Thêm một object mới vào mảng cards.
{
  type: "photo",
  src: "photos/lop-12a1-anh-moi.jpg",
  caption: "Một tấm ảnh mới"
}
07

Ví dụ đầy đủ từ đầu đến cuối

Ví dụ bạn muốn thêm ảnh mới tên ao-trang-san-truong.jpg vào kỷ niệm Ngày khai giảng:

  1. Upload file ao-trang-san-truong.jpg vào thư mục photos/.
  2. Mở file data.js.
  3. Tìm đến mục title: "Ngày khai giảng".
  4. Trong mảng cards, thêm đoạn này:
{
  type: "photo",
  src: "photos/ao-trang-san-truong.jpg",
  caption: "Sân trường hôm ấy"
}
  1. Kéo xuống cuối trang GitHub và nhập mô tả commit, ví dụ: Thêm ảnh mới cho mục Ngày khai giảng.
  2. Bấm Commit changes.
  3. Chờ 1-2 phút, sau đó mở lại website.
08

Checklist mỗi lần cập nhật

  • Tên file trong data.js phải giống đúng tên file trong repo.
  • Ảnh phải nằm trong photos/.
  • Video phải nằm trong videos/.
  • Sau khi sửa nhớ bấm Commit changes.
  • Nếu website chưa đổi ngay, chờ thêm 1-2 phút để GitHub Pages cập nhật.

Trang quản trị

Nhập mật khẩu để mở bảng điều khiển và cập nhật website kỷ niệm.

Bảng điều khiển kỷ niệm

Điền nội dung, kéo thả ảnh hoặc video, rồi bấm lưu để website công khai cập nhật ngay trên GitHub Pages.

Ngôn ngữ: Tiếng Việt

Cài đặt website

Thông tin này sẽ hiện ở phần đầu trang chính.

Quản lý kỷ niệm

Mỗi kỷ niệm là một cụm ảnh hoặc video. Bạn có thể thêm tối đa 6 ô media cho mỗi mục.

Lưu và cập nhật website Dữ liệu sẽ được lưu trong trình duyệt bằng localStorage.

Hướng dẫn xuất file ảnh/video

  1. Tải file ảnh/video xuống máy.
  2. Upload vào thư mục /photos/ hoặc /videos/ trên GitHub.
  3. Bấm Commit changes.
  4. Website tự cập nhật sau 1-2 phút.
✅ Đã lưu! Website đã được cập nhật.
